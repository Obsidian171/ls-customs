## Как использовать фото

### Для сотрудников
1. Загрузите фото на хостинг (Imgur, Cloudinary, Supabase Storage)
2. Скопируйте URL
3. В админ-панели добавьте URL в поле `photo_url`

### Для галереи
1. Загрузите фото тюнинга
2. Добавьте через админку с URL
3. Укажите категорию (wheels, hydraulics, paint, body_kit)

### Для услуг
1. Загрузите иконку/фото услуги
2. Добавьте URL в поле `image_url`
3. Укажите категорию (repair, tuning, paint, diagnostics)
